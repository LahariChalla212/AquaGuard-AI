# AquaGuard AI 

A Confidence-Aware Water Potability Prediction System built using Machine Learning and Flask.

## Features

- Water Potability Prediction
- Confidence Score Analysis
- Water Quality Parameter Alerts
- Interactive Web Interface
- Random Forest Machine Learning Model

## Technologies Used

- Python
- Flask
- Scikit-Learn
- HTML
- CSS
- Machine Learning

## Dataset

Water Potability Dataset from Kaggle.

## Project Overview

AquaGuard AI predicts whether water is safe for drinking based on multiple water quality parameters such as:

- pH
- Hardness
- Solids
- Chloramines
- Sulfate
- Conductivity
- Organic Carbon
- Trihalomethanes
- Turbidity

The system also provides confidence scores and parameter alerts for better decision-making.
